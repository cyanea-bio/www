Get started in seconds

Install the CLI, pull datasets, and run analyses — all from the command line. Or use our Python SDK for seamless integration with your notebooks.

Read the Docs → Create an Account
CLI Python Rust 
# Install Cyanea CLI
$ curl -fsSL https://cyanea.bio/install | sh

# Pull a dataset
$ cyanea pull broad-institute/gnomad-v4.1

# Run a local analysis
$ cyanea run align --ref hg38 --reads sample.fastq.gz

# Push results back
$ cyanea push my-org/variant-calls
