How it works

From raw data to shared, reproducible research in three simple steps.

1

Upload your data

Drag and drop files or use the CLI. FASTA, FASTQ, VCF, CSV, images — we handle it all.

2

Organize and analyze

Create projects, add metadata, run analysis pipelines. Everything is versioned automatically.

3

Share and discover

Publish datasets, fork protocols, cite and credit. Build on each other's work.

terminal 
$ cyanea upload --project BRCA-cohort genome_*.fasta
Uploading 24 files (48.2 GB)... done in 12.4s

$ cyanea analyze --pipeline variant-calling
